Case Study

How Dutch rational principles guide real-world design decisions: from concept to execution.

The Brief

Design a modular poster system for a design conference. The challenge: create visual variety using a single grid, limited color palette, and strict typography rules. No decoration—only structure and communication.

The Process

  1. Define the grid: 12-column grid with 8pt baseline for alignment precision.
  2. Choose palette: Ink, paper, red, blue, yellow—borrowed from De Stijl and Swiss tradition.
  3. Set typography: Single sans-serif, uppercase headings, consistent sizes.
  4. Iterate compositions: Six variations using identical elements in different arrangements.
  5. Test across media: Print posters at actual size, then translate to web.

The Outcome

A cohesive visual identity that feels both systematic and flexible. Each poster is recognizably part of a family, yet distinct in hierarchy and emphasis. The system scales from a poster to a business card without loss of clarity.

Key insight: Constraint breeds creativity. A strict system forces designers to explore nuance within boundaries—exactly what Dutch rational modernism demands.

About this repository

Built as a coded sketchbook for Dutch Rational Modernism, combining lessons from Bauhaus exercises and Swiss-influenced layout systems.

Intent

The code is meant as a starting point for experiments: swap typefaces, tweak the grid, or extend the gallery to your own poster series.

Usage

Clone the repo, adjust the variables in the CSS file, and fork the layout for case studies, portfolios, or design documentation.
