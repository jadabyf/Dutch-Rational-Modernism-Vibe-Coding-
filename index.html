Core principles

A rational system: minimal, repeatable, and readable across print and screen.

Grid Discipline

Every element is aligned to a strict grid, creating structure, rhythm, and visual hierarchy.

Typographic Order

Sans-serif faces, controlled scales, and clear leading support reading before style.

Geometric Reduction

Rectangles, lines, and blocks of color are reduced to essentials, echoing De Stijl.

Function First

Composition follows communication: each decision is justified by clarity and function.

About this repository

Built as a coded sketchbook for Dutch Rational Modernism, combining lessons from Bauhaus exercises and Swiss-influenced layout systems.

Intent

The code is meant as a starting point for experiments: swap typefaces, tweak the grid, or extend the gallery to your own poster series.

Usage

Clone the repo, adjust the variables in the CSS file, and fork the layout for case studies, portfolios, or design documentation.
